# 🌐 InfraSphere-3D

A next-generation 3D interactive learning platform designed for Infrastructure Planning & Management. The project transforms traditional study material into an immersive visual experience using animated components, interactive cards, case studies, risk frameworks, PPP models, and infrastructure management concepts.

---

## 🚀 Features

### 🎯 Interactive Learning
- 3D animated user interface
- Interactive flip cards
- Smooth navigation experience
- Scroll-based animations

### 📊 Infrastructure Planning
- Project Viability Analysis
- NPV, IRR, BCR & Payback Methods
- Feasibility Assessment
- PPP Frameworks

### ⚠️ Risk Management
- Risk Matrix Visualization
- EMV Calculations
- PFCEO Framework
- Risk Allocation Models

### 🏗 Asset Management
- ISO 55000 Concepts
- Asset Lifecycle Planning
- Depreciation Models
- Asset Valuation Methods

### 🤖 Smart Infrastructure
- IoT Integration Concepts
- Digital Twin Models
- BIM Framework
- AI-Based Infrastructure Monitoring

### 📚 Case Studies
- Dabhol Power Project
- Airport Privatization
- Vizhinjam Port
- Airtel vs Jio
- Indian Railways
- AirAsia India

### 🧮 Numerical Guide
- NPV Formula
- IRR Calculation
- BCR Analysis
- EMV Formula
- Risk Scoring
- Depreciation Methods
